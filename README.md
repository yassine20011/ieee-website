# IEEE Student Branch EMSI Marrakesh Website

Welcome to the official website of the IEEE Student Branch at EMSI Marrakesh. This platform is designed to showcase our team, our achievements, our cells, and our upcoming events.

## 🚀 Features

- **Dynamic Homepage**: Overview of our mission, team, and activities.
- **Team & Supervisors**: Dedicated pages to meet our leadership and mentors.
- **Events & Achievements**: Track our history and stay updated on upcoming competitions and workshops.
- **Performance Optimized**: Built with modern web standards, featuring lazy loading and automated asset optimization.

## 🛠️ Technology Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 7](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 3](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/)

## ⚡ Quick Start

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/ieee-student-branch-emsi-marrakesh/ieee-website.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
    *(Note: This is aliased to `vercel dev` to support serverless functions for forms and local `.env` variables)*
